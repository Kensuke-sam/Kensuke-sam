## ⚡ Philosophy

### `AI-native developer mindset`

- **Ship before the idea cools down**  
  Momentum matters. A rough release teaches more than a perfect plan sitting in drafts.

- **Use AI as production infrastructure**  
  Claude Code, Codex, ChatGPT, and Gemini are part of how I think, test, draft, and build.

- **Memorization is no longer the moat**  
  Judgment, iteration speed, and leverage design matter more than raw recall.

- **Small tools can create disproportionate value**  
  I like products that solve narrow problems well and earn their place through utility.

- **Creative systems beat static skill sets**  
  The point is not to fit one label. The point is to keep combining tools, ideas, and mediums.
